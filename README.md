# Coral Nursery Aerial Imagery Dataset

**Contains the drone imagery dataset collected for the GIS-based coral structure placement estimation system.**

---

## Overview
This repository contains the drone imagery dataset collected for the **GIS-based coral structure placement estimation system**.  
The dataset includes **high-resolution aerial images** captured over coral nursery sites for mapping and spatial analysis of coral nurseries.
